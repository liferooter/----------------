# Cuban Jumper
Simple game written in Python & PyGame created by me from nothing-to-do in Cuba

## How to run?
Run module `app`:
```bash
python -m app
```

## Requirements
- Python 3.9 (Python version that I use)
- PyGame (`python -m pip install pygame`)
